# STORE ITEMS LOCATION
This app use to show location production in map.

## Version
- Java 17.0.1
- Netbeans 12.0

## How to run
1. Copy config.exampe.txt to config.txt
2. Add library mysql-connector-java-5.1.49.jar
3. Run App.java
